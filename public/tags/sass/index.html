Sass for Designers

Posted: 28 December 2012

Despite saying that I wanted to avoid writing post about development, I wanted to write something about Sass. Excuse me, what is Sass? Sass is a CSS pre-processing language; it’s a slightly different way of writing CSS which can then be processed by a tool that spits out fully-working CSS. It’s like a kind of short-hand that adds in some handy features that aren’t available in CSS. If you want to know how to use Sass in your workflow, try my post on Sass for Designers — The Setup. Read Sass for Designers.
